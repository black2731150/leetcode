# 链表

### 数组和链表有什么区别


| 特性         | 数组           | 链表          |
| ----------- | ------------- | ------------- |
| **内存地址** | 连续的内存     | 非连续的内存   |
| **长度**     | 固定的        | 动态变化的     |
| **增删效率** | 较低（需要移动其他元素） | 较高（只需修改指针） |
| **查询效率** | 高（可直接通过索引访问） | 低（需遍历查找）  |
| **数据访问** | 随机访问       | 顺序访问       |
